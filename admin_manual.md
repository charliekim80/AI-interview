# TecAce AI Interview 운영 매뉴얼 (HR Admin용)

본 매뉴얼은 TecAce AI Interview 플랫폼을 사용하는 채용 담당자(Admin)를 위한 가이드입니다. 지원자 등록부터 결과 분석까지의 전 과정을 설명합니다.

---

## 1. 로그인 및 접속
- **URL**: `https://[Your-Domain]/admin`
- **계정**: 관리자로부터 부여받은 이메일과 비밀번호로 로그인합니다.

> [!NOTE]
> 보안을 위해 공용 PC 이용 시 반드시 로그아웃을 수행해 주세요.

---

## 2. 대시보드 (Dashboard)
대시보드에서는 전체 채용 현황을 한눈에 파악할 수 있습니다.

- **핵심 지표**: 전체 지원자 수, 면접 완료 건수, 진행 중인 면접, 평균 AI 점수.
- **지원자 목록**: 각 지원자의 상태(등록됨, 초대됨, 진행 중, 완료)를 실시간으로 확인합니다.
- **액션 버튼**:
  - `결과 뷰어`: 면접이 완료된 지원자의 상세 분석 내용을 확인합니다.
  - `링크 복사`: 지원자에게 발송할 면접 초대 링크를 복사합니다.
  - `면접 결과 초기화`: 기술적 문제 등으로 재면접이 필요한 경우 기존 데이터를 삭제하고 리셋합니다.

> [!TIP]
> **[스크린샷 가이드]**: 대시보드 전체 화면을 캡처하여 상단 지표 영역과 하단 목록 영역을 표시하세요.

---

## 3. 포지션 관리 (Job Position)
새로운 채용 공고를 등록하거나 수정합니다.

1. **상세 내용 입력**: 직무 제목, 팀/부서, 핵심 기술 스택, 담당 업무(JD)를 상세히 입력합니다.
2. **저장**: 저장된 JD는 AI가 면접 질문을 생성할 때 가장 중요한 참고 자료가 됩니다.

---

## 4. 지원자 등록 및 질문 생성 (Candidates)
가장 핵심적인 단계로, 지원자별 맞춤형 질문을 생성합니다.

1. **정보 입력**: 이름, 이메일, 전화번호와 함께 **이력서(PDF/DOCX)**를 업로드합니다. (최대 3개 합계 10MB)
2. **AI 질문 생성**: `AI 질문 생성` 버튼을 클릭하면 AI가 이력서와 JD를 분석하여 10개의 맞춤형 질문을 만듭니다.
3. **질문 검토 및 수정**:
   - 생성된 질문 중 면접에 사용할 질문을 선택합니다.
   - 각 질문의 텍스트를 직접 수정하거나, '심층 질문(꼬리 질문)' 활성화 여부를 결정합니다.
4. **링크 생성**: `면접 초대링크 생성`을 클릭하여 고유 URL을 발급받습니다.

> [!IMPORTANT]
> **심층 질문(Follow-up)**: 지원자의 답변이 부족할 경우 AI가 실시간으로 추가 질문을 던지는 기능입니다. 변별력이 필요한 문항에 활성화하세요.

---

## 5. 면접 결과 확인 (Interview Result)
면접이 완료된 지원자의 역량을 AI가 정밀 분석합니다.

- **종합 점수 및 추천**: 0~100점 사이의 점수와 'Highly Recommended' 등의 추천 등급을 제시합니다.
- **종합 평가 및 강점/개선점**: HR 담당자가 판단을 내리기 쉽도록 텍스트 요약을 제공합니다.
- **질문별 상세 분석**: 각 질문 카드는 아래 규칙으로 표시됩니다.
  - `Q1`, `Q2` … : 메인 질문 (일반 질문)
  - `Q1-1`, `Q1-2` : 심층질문(꼬리질문) — 메인 질문과 세트로 표기되며, 카드 상단에 🔗 **심층질문** 배지가 표시됩니다.
  - **질문 내용**: 지원자에게 실제로 전달된 원문 그대로 표시됩니다 (요약 없음).
  - **Score**: 모든 카드의 점수 배지가 우측 동일 열에 정렬되어 한눈에 비교 가능합니다.
- **Excel 다운로드**: 보고용 데이터가 필요한 경우 Excel 파일로 내보낼 수 있습니다.
  - 희망 연봉 문항 포함 여부를 선택할 수 있습니다.
  - **유형** 컬럼: 각 행이 '일반' 또는 '🔗 심층질문'으로 구분되어 표시됩니다.

> [!TIP]
> 심층질문(꼬리질문)은 지원자의 답변이 부족할 경우 AI가 실시간으로 자동 생성한 추가 질문입니다. 면접 질문 설정 시 2~7번 문항에서 활성화하면 변별력 향상에 효과적입니다.

---

## 6. 설정 (Settings)
- **OpenAI API Key**: AI 분석 및 질문 생성을 위한 API 키를 관리합니다.
- **분류(Category) 설정**: 조직 내의 부서나 부문의 목록을 관리하여 지원자 등록 시 선택 항목으로 사용합니다.

---

## FAQ 및 트러블슈팅
- **지원자가 링크에 접속할 수 없다고 합니다**: 링크가 만료되지 않았는지, 혹은 지원자의 브라우저(Chrome 권장) 설정을 확인하세요.
- **AI 질문 생성이 너무 오래 걸립니다**: 이력서 파일이 너무 크거나 텍스트가 복잡할 경우 최대 30초 정도 소요될 수 있습니다.
- **동결/화이트스크린 발생**: 인터넷 연결을 확인하고 페이지를 새로고침(`F5`) 하세요. 데이터는 서버에 안전하게 저장됩니다.
